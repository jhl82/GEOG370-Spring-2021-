In this project I learned how to display images in different projections

Describe in your own words how you displayed the map in different projections using QGIS

In order to display the map in different projections, I first opened up a shape file of a regular 2-D map in vector format. Once opened, you can go down to the corner and there is a button that says ESRI:. If you click this button it will open up a menu that gives you many different map projections to choose from.
